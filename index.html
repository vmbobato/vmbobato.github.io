Here are some projects I have worked on:

  • Engineering Capstone (Academic) - As a project manager I led the development of a laboratory bench with switches, routers, and firewalls to simulate a Supervisory Control and Data Acquisition (SCADA) system in energy utility companies. The lab bench included a custom Remote Terminal Unit (RTU) built with a Raspberry Pi that connected analog and digital sensors and actuators. The data from the sensors and the status of actuators are sent to a Graphical User Interface (GUI), where the data is displayed. The project used an industrial control system (ICS) protocol called Distributed Network Protocol 3 (DNP3) to establish network connection between the HMI and RTU to send commands and collect measurements. The project used Python 3.9 for the embedded code and the MCP3008 10-bit ADC with an SPI interface. The DNP3 protocol was implemented with the OpenDNP3 C++ library.
  • Machine Learning Based Firewall (Academic) - Implemented a Python-based Intrusion Detection System (IDS) using the IoT-23 dataset to improve real-time incident response. My goal was to create a system that could analyze network traffic and accurately identify malicious packets. To achieve this, I trained a Random Forest Classifier using supervised learning with the IoT-23 dataset, allowing it to recognize patterns in the data and differentiate between normal and suspicious activity.
  • ShellBridge (Personal) - Developed a reverse shell application, enabling remote server interaction. Facilitated secure file transfers with download and upload functionalities using SSL/TLS, demonstrating skills in networking and cybersecurity. This project was created to facilitate the file transfer between virtual machines.
  • Personal Password Manager (Personal) - Developed a personalized password manager leveraging Python programming. Implemented algorithms to generate robust passwords, while efficiently storing them in a secure MySQL database using a MySQL Python API, underscoring skills in encryption, data protection, and database management.
